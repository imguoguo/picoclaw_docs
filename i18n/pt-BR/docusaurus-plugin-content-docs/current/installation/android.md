# Instalar no Android

O PicoClaw disponibiliza um pacote de instalação para Android, permitindo que você instale o PicoClaw em dispositivos Android.
Este guia irá orientá-lo na instalação do PicoClaw no Android.

## Instalação

### Baixar o Pacote de Instalação

1. Acesse o [Site Oficial do PicoClaw](https://picoclaw.io/download/android/) para baixar o pacote de instalação para Android

![Download](/img/installation/android0.png)

2. O pacote APK de instalação baixado

![Instalar](/img/installation/android1.png)

## Iniciar o PicoClaw

Após a instalação, toque no ícone do PicoClaw para abrir o aplicativo e, em seguida, toque no botão "Iniciar Serviço" para iniciar o serviço.
![Iniciar](/img/installation/android2.png)

Assim que o serviço estiver em execução, você poderá acessar a interface Web do PicoClaw a partir da página de configurações.
Alternativamente, abra o navegador e acesse o seguinte endereço para acessar a interface Web do PicoClaw:

```text
http://127.0.0.1:18800
```
![WebUI](/img/installation/android3.png)

- Se você precisar acessar a interface Web do PicoClaw externamente, ative o Modo Público nas configurações.
![Configurações](/img/installation/android4.png)
